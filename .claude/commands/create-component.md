---
description: Neue Komponente mit HTML, CSS und JS erstellen
argument-hint: [komponenten-name] [typ: button|card|modal|form|nav|hero]
---

Erstelle eine neue Komponente namens `$1` vom Typ `$2`.

Die Komponente soll enthalten:

1. **HTML-Struktur**
   - Semantisches Markup
   - BEM-Klassennamen
   - ARIA-Attribute
   - Data-Attribute für JS

2. **CSS-Styling**
   - CSS-Variablen für Theming
   - Mobile-First Responsive
   - Hover/Focus/Active States
   - Animationen (prefers-reduced-motion beachten)

3. **JavaScript (falls nötig)**
   - Event Handling
   - State Management
   - Accessibility (Keyboard, Screen Reader)

4. **Dokumentation**
   - Verwendungsbeispiel
   - Props/Optionen
   - Varianten

Erstelle die Dateien in der entsprechenden Projektstruktur.
