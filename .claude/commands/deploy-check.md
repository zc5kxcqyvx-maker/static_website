---
description: Pre-Deployment Checkliste durchgehen
---

Führe eine Pre-Deployment Prüfung durch:

## Build & Assets
- [ ] Build läuft fehlerfrei
- [ ] Keine Console-Errors/Warnings
- [ ] Assets minifiziert (CSS, JS)
- [ ] Bilder optimiert (WebP, komprimiert)
- [ ] Favicon und App-Icons vorhanden

## SEO & Meta
- [ ] Alle Seiten haben Title-Tags
- [ ] Meta-Descriptions vorhanden
- [ ] robots.txt konfiguriert
- [ ] sitemap.xml generiert
- [ ] Canonical URLs gesetzt

## Performance
- [ ] Lighthouse Score > 90
- [ ] Lazy Loading für Bilder
- [ ] Kritisches CSS inline
- [ ] Fonts optimiert (preload, swap)

## Security
- [ ] HTTPS konfiguriert
- [ ] Security Headers gesetzt
- [ ] Keine sensiblen Daten im Code
- [ ] Dependencies aktuell

## Accessibility
- [ ] WCAG 2.1 AA konform
- [ ] Fokus-Styles sichtbar
- [ ] Alt-Texte vollständig

## Funktionalität
- [ ] Alle Formulare getestet
- [ ] 404-Seite vorhanden
- [ ] Analytics eingebunden
- [ ] Cookie-Banner (falls nötig)

Markiere was fehlt und was erledigt ist.
